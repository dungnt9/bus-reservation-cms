# Bus Management System (CMS)

A comprehensive bus management system built with Vue.js that helps manage bus routes, schedules, tickets, and passenger information.

---

## Features

### User Management

- Admin management
- Driver management
- Assistant management
- Customer management

### Route Management

- Create and manage bus routes
- Set ticket prices
- Track distances and estimated durations

### Trip Management

- Schedule trips
- Assign drivers and vehicles
- Track departure/arrival times
- Monitor seat availability

### Ticket & Booking

- Interactive seat selection
- Invoice generation
- Payment processing (Cash/Card)
- Booking status tracking

### Vehicle Management

- Track vehicle information
- Manage seat configurations
- Monitor vehicle availability

### Reporting & Analytics

- Revenue tracking
- Trip completion rates
- Customer trends
- Performance metrics

---

## Tech Stack

- **Vue.js 3**
- **Vue Router**
- **Pinia (State Management)**
- **Bootstrap Vue**
- **Chart.js**
- **Axios**
- **FontAwesome**

---

## Getting Started

### Install dependencies:

```sh
npm install
```

### Run development server:

```sh
npm run dev
```

### Build for production:

```sh
npm run build
```
